Aseba est un environnement de programmation pour le robot Thymio II.
VPL est un composant d'Aseba qui permet une programmation graphique
en assemblant des blocs de couleurs. Cette archive contient un tutoriel
sur l'utilisation du robot Thymio II avec VPL. Pour plus d'information
sur le Thymio II et Aseba, voir : https://aseba.wikidot.com/.

Il y a quatre documents et deux répertoires dans cette archive :

thymio-vpl-tutorial-fr.pdf:             Le tutoriel
thymio-vpl-tutorial-answers-fr.pdf:     Les réponses aux questions
thymio-vpl-ref-card-fr.pdf:             Une page A4 recto-verso de référence
thymio-vpl-folding-ref-card-fr.pdf:     Une carte de référence pliable

programs:     Les programmes Aseba/VPL pour les exemples du tutoriel
